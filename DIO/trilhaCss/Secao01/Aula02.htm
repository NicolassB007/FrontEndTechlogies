Seletores CSS

No CSS nós possuímos os seletores por Id

Iniciei minha Trilha de CSS!

  • Seletor por Tipo: Busca elementos por uma Tag HTML específica
  • Seletor por Id (#): Busca um elemento através do atributo "id"
  • Seletor por Classe (.): Busca um elemento através do atributo "class"
  • Seletor Universal (*): Seleciona todos os elementos HTML
  • Seletor de Atributo ([atrib]): Seleciona elementos que possuem um atributo específico em nosso documento HTML e conseguimos também buscar atributos com um valor específico.
